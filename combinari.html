Combinari

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    if (k > n) {
        cout << "Wrong input!\n";
        return 0;
    }
    vector<int> elements(k + 1);

    int index = 0;
    while (!elements[0]) {
        for (int i = index; i < k; ++i) { // recalculates the elements after
            elements[i + 1] = elements[i] + 1;
        }
        index = k;
        for (int i = 1; i <= k; ++i) { // show elements
            cout << elements[i] << ' ';
        }
        cout << '\n';
        while (elements[index] == n - k + index && index) { // find element to be changed
            --index;
        }
        ++elements[index];
    }
    return 0;
}

Acest algoritm primeste numerele n si k si afiseaza toate combinarile de n luate cate k.

Cum am gandit acest algoritm?

Citim numerele n si k. Avem o bucla care afiseaza fiecare combinare posibila si este formata din 3 parti. Ne folosim de o variabila ca sa stim de unde trebuie schimbam elementele. In prima parte actualizam elementele de dupa acea variabila care initial este egala cu 0; Dupa actualizam variabila la valoarea lu k. Afisam elementele cu un for, apoi calculam pozitia variabilei la variabila cea mai din dreapta care mai poate fi marita. In cazul in care al doilea element din vector a ajuns la capat (primul afisat) va creste primul element din lista. In acest caz inseamna ca s-au afisat toate combinarile si iesim din bucla.

La ce foloseste acest algoritm?

Acest algoritm este de timp backtracking si ne ajuta sa folosim viteza mare a calculatorului pentru a lucra cu o multime mare de elemente care au anumite proprietati.

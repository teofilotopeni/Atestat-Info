Cautarea binara

#include <iostream>
using namespace std;

int main() {
    int n, elem[1000000];
    cin >> n;
    for (int i = 0; i < n; ++i) {
        cin >> elem[i];
    }
    int cautat, a = 0, b = n, mediu;
    cin >> cautat;
    while (elem[a] != cautat && a < b - 1) { // cautam elementul in intervalul a b
        mediu = (a + b) / 2;
        if (elem[mediu] > cautat) {
            b = mediu;
        }
        else {
            a = mediu;
        }
    }
    if (elem[a] == cautat) {
        cout << a + 1 << '\n';
    }
    else {
        cout << "Nu sa gasit!\n";
    }
    return 0;
}

Acest algoritm primeste un vector de elemente sortat si cauta un element dat.

Cum am gandit acest algoritm?

La inceput citim vectorul si elementul cautat. Apoi ne folosim de o bucla pentru a cauta elementul. In bucla folosim doua variabile pentru a retine intervalul in care cautam. La fiecare pas comparm mijlocul intervalului cu elementul cautat. Daca elementul este mai mare actualizam intervalul la juamtatea din dreapta, altfel la jumatatea din stanga. Cand intervalul ajunge la un singur element iesim din bucla. Daca elementul din interval este egal cu cel cautat ii afisam pozitia altfel spunem ca nu l-am gasit.

La fiecare pas se injumatateste intervalul astfel cautarea binara are complexitatea O(log n) si aici mai avem si citirea elementelor O(n).

La ce foloseste acest algoritm?

Acest algoritm este folosit pentru a cauta anumite elemente in baze foarte mari de date foarte rapid.
